# Maze Runner UI (SFML)

## 🎮 Description
A simple Maze Runner game built using C++ and SFML with a graphical UI.

## 🚀 Features
- Grid-based maze
- Player movement (WASD)
- Wall collision
- Exit detection
- Clean UI using SFML

## 🛠 Requirements
- SFML installed

## ▶️ Run
g++ main.cpp -o game -lsfml-graphics -lsfml-window -lsfml-system
./game
