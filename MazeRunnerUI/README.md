# 🎮 Maze Runner (C++ Console Game)

A simple yet interactive **Maze Runner game** built using C++ with an enhanced console-based UI.

---

## 🚀 Features

* 🧩 Grid-based maze system
* 🎮 Real-time player movement (WASD controls)
* 🚧 Wall collision detection
* 🎯 Exit detection with win condition
* 🎨 Colored console UI (better UX)
* 📊 Move counter system

---

## 🎮 Controls

| Key | Action       |
| --- | ------------ |
| W   | Move Up ↑    |
| S   | Move Down ↓  |
| A   | Move Left ←  |
| D   | Move Right → |

---

## ⚙️ How to Run

### 1. Compile

```bash
g++ main.cpp -o game.exe
```

### 2. Run

```bash
./game.exe
```

## 🧠 Concepts Used

* Object-based thinking (Player, Grid system)
* 2D arrays (maze representation)
* Conditional logic & movement handling
* Console UI design using colors
* Real-time input handling (`_getch()`)
